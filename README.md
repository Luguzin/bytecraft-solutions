# ByteCraft Solutions

[![Website](https://img.shields.io/badge/Website-ByteCraft%20Solutions-blue?style=for-the-badge )](https://luguzin.github.io/bytecraft-solutions/ )

Página web institucional para uma startup fictícia de tecnologia focada em desenvolvimento web.

## 📋 Descrição

ByteCraft Solutions é uma startup de tecnologia que oferece soluções inovadoras em desenvolvimento web. O site apresenta informações sobre a empresa, seus serviços, equipe e formas de contato.

## 🎯 Conteúdo do Site

- **Início**: Seção hero com imagem criativa de tecnologia
- **Sobre**: Informações sobre a empresa e sua visão
- **Serviços**: Três serviços principais (Design Web, Desenvolvimento Frontend, Consultoria Digital)
- **Equipe**: Tabela com informações do fundador
- **Competências**: Linguagens de programação utilizadas
- **Portfólio**: Projetos desenvolvidos
- **Contato**: Informações de contato e links

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Imagens otimizadas

## 📁 Estrutura do Projeto

```
bytecraft_solutions/
├── index.html          # Página principal
├── css/
│   └── style.css       # Estilos do site
└── README.md           # Este arquivo
```

## 🚀 Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Navegue pelas seções usando o menu no topo
3. Clique nos links para mais informações

## 📱 Responsividade

O site é totalmente responsivo e funciona em:
- Computadores (desktop)
- Tablets
- Celulares (mobile)
